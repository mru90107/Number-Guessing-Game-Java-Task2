# 🎯 Number Guessing Game (Java) - Task 2

## 📌 Overview
This project is a console-based Number Guessing Game developed in Java as part of my internship task.  
The program generates a random number between 1 and 100, and the user has to guess it with hints provided after each attempt.

---

## ✨ Features
- Random number generation (1–100)
- User input handling
- Input validation (only numbers allowed)
- Range checking
- Hint system (Too High / Too Low)
- Attempt counter
- Play again option

---

## 🧠 Concepts Used
- Java Programming
- Loops (while, do-while)
- Conditional Statements (if-else)
- Scanner Class
- Random Class
- Input Validation

---
